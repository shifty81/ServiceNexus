## 🚀 Quick Start

### Installation
```bash
# Clone repository
git clone https://github.com/shifty81/FieldForge.git
cd FieldForge

# Install all dependencies
npm install
cd client && npm install && cd ..

# Start development
npm run dev
```

### Access
- Frontend: http://localhost:3000
- Backend: http://localhost:3001

### First Steps
1. Register an account
2. Go to "AI Upload" → "📸 Use Camera"
3. Capture any document
4. Watch AI create your form!
